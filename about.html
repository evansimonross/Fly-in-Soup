About Fly-in-Soup

Fly-in-Soup is a tool for New Yorkers to easily access the health inspection data on restaurants in their area. You can search by restaurant name, current location, set address, or ZIP code, and you will be treated to a map of places where you are and aren't likely to find a Fly in your Soup!

This web application was designed with love by Katherine He, Jano Roze & Evan Simon Ross. It is the first project for the first cohort of the Columbia Full Stack Web Development Coding Bootcamp.

This project is open source! Feel free to check out our GitHub Repository.

Thanks for using Fly-in-Soup!
